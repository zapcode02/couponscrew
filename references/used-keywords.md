# Used Keywords

> Primary keywords already targeted on Couponscrew.
> **Rule:** before picking a primary keyword for a new blog post or page, check this file first. Never reuse a primary — that causes keyword cannibalisation.

---

## Keyword-selection rules

1. **Primary keyword must have real search volume** — verify using Ahrefs, Semrush, or Google Keyword Planner before assigning.
2. **Check this file first** — never reuse a primary already listed below.
3. **Secondary/cluster keywords:** find same-intent variations of the primary. Would someone searching this phrase want to land on the same page? If yes, it's a secondary. If no, it belongs in a different cluster.
4. **Long-tail with coupon modifiers are highest intent.** Keywords like "[brand] coupon code", "[brand] promo code today" convert far better than generic category terms.
5. **Mark the source** for each keyword — CSV, tool, or invented — so future writers know what's verified.
6. **Coupon pages need 1,500+ words of unique content** to be worth indexing. Under that, consolidate onto a category hub page.

## Couponscrew keyword framework

Use these categories as a starting framework when researching new primaries. These are the proven structures that rank for coupon sites globally (RetailMeNot, Honey, CashKaro, GrabOn).

### Tier 1 — Highest intent (brand + coupon)
```
[brand name] coupon code
[brand name] promo code
[brand name] discount code
[brand name] offer today
[brand name] coupon code today
```
Target one brand per page. These are your highest-converting pages.

### Tier 2 — Sale event keywords
```
big billion day deals [year]
amazon great indian festival coupons
myntra end of reason sale codes
flipkart sale date [year]
[platform] festive sale offers
```

### Tier 3 — Savings tips & how-to (blog content)
```
how to save money online shopping india
how to stack coupons india
best cashback apps india 2026
bank card offers online shopping india
how to get [brand] discount
does [brand] coupon code work
```

### Tier 4 — Category hub pages (3,500+ words each)
```
fashion coupon codes india
electronics deals india
food delivery promo codes india
travel booking discount codes india
web hosting deals india
beauty coupons india
grocery discount codes india
```

### Tier 5 — Quick commerce (emerging, low competition)
```
blinkit promo code
zepto discount code
instamart coupon code
swiggy instamart offers
quick delivery discount india
```

## Keyword research tools to use

- **Ahrefs** — best for KD scores and competitor gap analysis
- **Semrush** — good for tracking coupon site competitors (CashKaro, GrabOn)
- **Google Keyword Planner** — free, reliable volume estimates for Indian market
- **Google Search "People Also Ask"** — free source for secondary and long-tail keywords

---

## Workflow for each new post

1. Pick a primary from the framework categories above
2. Verify volume and KD in Ahrefs or Semrush
3. Check it's not already in the Active Primaries list
4. Find 3–5 same-intent secondaries
5. Add the new entry to Active Primaries before writing
6. Target the primary in: page title, H1, first 100 words, one H2, meta description
7. Update "Used on page" after publishing
